# Agile Documentation

## Project Title
Visa Processing Time Prediction

## Methodology
This project follows Agile methodology, where development is divided into multiple iterations (sprints) to ensure continuous improvement and delivery.

## Sprint Planning

### Sprint 1: Data Collection & Preprocessing
- Collected visa dataset
- Cleaned missing values
- Prepared structured data

### Sprint 2: Exploratory Data Analysis (EDA)
- Visualized data patterns
- Identified important features
- Performed feature engineering

### Sprint 3: Model Development
- Built regression models
- Trained and tested models
- Evaluated performance

### Sprint 4: Deployment
- Developed Streamlit web app
- Integrated model with UI
- Tested final application

## User Stories
- As a user, I want to enter visa details so that I can predict processing time.
- As a system, I should provide accurate predictions based on historical data.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
